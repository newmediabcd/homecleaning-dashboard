# 크린토피아 홈클리닝 Weekly Report 코멘트 작성 가이드

---

## 작성 원칙

- 보고서체 단문 (존댓말 금지)
- 키워드는 반드시 [ ] 대괄호 표기, 유형은 ( ) 괄호로 병기
- 수치는 구체적으로 (만원 단위, % 단위, 전환 건수)
- 목표 CPA: 30,000원 기준
- 전환 0건 키워드의 CPA는 "전환 미발생" 표기

---

## 키워드 유형 분류 기준

| 유형 | 판별 기준 | 예시 |
|---|---|---|
| 브랜드 | "크린토피아" 포함 | 크린토피아입주청소, 크린토피아청소 |
| 지역 | 지역명 포함 (서울, 부산, 강남, 수원 등) | 서울입주청소, 강남이사청소 |
| 메인 | 브랜드·지역 외 일반 청소 키워드 | 입주청소, 이사청소, 입주청소비용 |

---

## 시트별 코멘트 양식

### 공통 구성 (4줄)

| 줄 | 내용 |
|---|---|
| 1 | 주간 KPI 요약: 광고비 / 전환 / CPA — 전주 대비 증감 |
| 2 | 전환 상위 키워드 3~4개 (특성별 묶음: 메인/지역/브랜드) |
| 3 | 광고비 상위 키워드 5개 (줄바꿈 나열) |
| 4 | 특이사항 (선택) |

---

### 1. Summary (전체 요약)

**본문 구조 (2줄):**

| 줄 | 내용 |
|---|---|
| 1 | 네이버 SA 주간 광고비, 전환, CPA — 전주 대비 광고비 ±%, CPA 개선/상승 |
| 2 | 구글 SA 주간 광고비, 전환, CPA — 전주 대비 광고비 ±%, CPA 개선/상승 |

**작성 예시:**
```
네이버 SA 주간 광고비 142.3만원, 전환 48건, CPA 29,646원 — 전주(138.1만원, 전환 44건, CPA 31,386원) 대비 광고비 +3%, CPA 개선.
구글 SA 주간 광고비 81.5만원, 전환 22건, CPA 37,045원 — 전주(79.2만원, 전환 19건, CPA 41,684원) 대비 광고비 +3%, CPA 개선.
```

---

### 2. 네이버 SA · PC / MO

**본문 구조 (3줄):**

| 줄 | 내용 |
|---|---|
| 1 | 주간 KPI 요약 + 전주 대비 광고비 ±%, CPA 개선/상승 |
| 2 | 전환 상위 키워드 3~4개 (메인/지역/브랜드 특성별) |
| 3 | 광고비 상위 키워드 5개 (줄바꿈 나열) |

**작성 예시:**
```
광고비 78.2만원, 전환 31건, CPA 25,226원 — 전주(76.0만원, 전환 29건, CPA 26,207원) 대비 광고비 +3%, CPA 개선.
전환 상위 키워드
  메인) [입주청소] 전환 18건 / CPA 21,500원
  메인) [이사청소] 전환 5건 / CPA 17,000원
  브랜드) [크린토피아입주청소] 전환 4건 / CPA 15,500원
  지역) [서울입주청소] 전환 2건 / CPA 28,000원
광고비 상위 키워드
  1. [입주청소] 38.6만원 / 전환 18건 / CPA 21,500원
  2. [입주청소비용] 12.1만원 / 전환 미발생
  3. [이사청소] 8.5만원 / 전환 5건 / CPA 17,000원
  4. [크린토피아입주청소](브랜드) 6.2만원 / 전환 4건 / CPA 15,500원
  5. [서울입주청소](지역) 5.3만원 / 전환 미발생
```

---

### 3. 구글 SA · 브랜드 / 경쟁사 / 일반

**본문 구조 (3줄):**

| 줄 | 내용 |
|---|---|
| 1 | 주간 KPI 요약 + 전주 대비 |
| 2 | 전환 상위 키워드 3~4개 (메인/지역/브랜드 특성별) |
| 3 | 광고비 상위 키워드 5개 (줄바꿈 나열) |

**작성 예시 (구글 일반):**
```
광고비 31.5만원, 전환 8건, CPA 39,375원(목표초과) — 전주(33.2만원, 전환 6건, CPA 55,333원) 대비 광고비 -5%, CPA 개선.
전환 상위 키워드
  메인) [입주청소] 전환 5건 / CPA 28,400원
  메인) [이사청소] 전환 2건 / CPA 41,500원
  메인) [새집입주청소] 전환 1건 / CPA 32,000원
광고비 상위 키워드
  1. [입주청소] 14.2만원 / 전환 5건 / CPA 28,400원
  2. [이사청소] 8.3만원 / 전환 2건 / CPA 41,500원
  3. [입주청소비용] 4.1만원 / 전환 미발생
  4. [새집입주청소] 3.2만원 / 전환 1건 / CPA 32,000원
  5. [이사청소비용] 1.7만원 / 전환 미발생
```

## 자동 생성 항목 안내

`build_weekly.py` 실행 시 대시보드 코멘트는 자동으로 생성됩니다.

| 자동 생성 항목 | 설명 |
|---|---|
| 주간 KPI 요약 | 광고비, 전환, CPA 전주 대비 자동 비교 |
| 최다 전환 키워드 | 주차 내 전환 1위 키워드 + 유형 + CPA |
| 광고비 상위 5개 키워드 | spend 기준 상위 5개 + 전환 성과 |
