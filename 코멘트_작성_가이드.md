# 크린토피아 홈클리닝 Daily Report 코멘트 작성 가이드

---

## 작성 원칙

- 보고서체 단문 (존댓말 금지)
- 키워드는 반드시 [ ] 대괄호 표기
- 수치는 구체적으로 (만원 단위, % 단위)
- "대폭" 표현 금지 → "상승" "개선" "감소" "초과" 등 담백하게
- 목표 CPA: 30,000원 기준으로 판단
- 자동입찰은 주말에 설정 변경 없음 (주말 저소진 = 검색 수요 감소로 판단)

---

## ⚠️ 주말 3일 통합 규칙 (금·토·일)

> **데이터가 금요일·토요일·일요일에 해당하면, 단일 날짜가 아닌 금~일 3일 합산 기준으로 코멘트를 작성한다.**

### 적용 대상
Summary, 네이버 PC, 네이버 MO, 네이버 자동입찰, 구글 브랜드, 구글 경쟁사, 구글 일반 — **전 페이지 공통 적용**

### 제목 형식 (주말 버전)
```
{섹션명} — 주말 합산 ({시작일 MM/DD}~{종료일 MM/DD} 금~일)
```

### 핵심 작성 규칙

| 항목 | 평일 기준 | 주말 3일 통합 기준 |
|---|---|---|
| 비교 기준 | 전일 대비 | **전주 동일 주말(금~일) 합산 대비** |
| 광고비/전환/CPA | 당일 수치 | **금+토+일 합산** 수치 |
| 저소진 분석 | 평일 일평균 대비 | 전주 주말 합산 또는 평일 일평균 대비 |
| 자동입찰 | 전일 vs 전전일 | **금 vs 목(전전일)**, 토~일은 추이 설명 |

### Summary 제목 형식 (주말 버전)
```
주말 합산 ({MM/DD}~{MM/DD}) 요약 — 전주 주말({전주 금~일}) 대비
```

### 네이버 PC / MO 제목 형식 (주말 버전)
```
네이버 SA · PC — 주말 합산 ({MM/DD}~{MM/DD} 금~일)
```

### 작성 예시 (네이버 SA MO 주말 통합 버전)
```
[주말 합산] 광고비 190.4만원 / 전환 42건 / CPA 45,333원 — 전주 주말(광고비 228.7만원, CPA 55,777원) 대비 광고비 -17% 감소, CPA 18% 개선.

주말 3일 평균 광고비 63.5만원 — 평일 일평균(79.9만원) 대비 -21% 저소진. 주말 검색 수요 감소에 따른 구조적 패턴으로 판단.

[MO 입주청소] 주말 합산 광고비 52.3만원, 전환 9건 (CPA 58,111원). [MO 이사청소] 전환 3건 (CPA 38,500원). [MO 입주청소가격] 전환 0건 주말 지속.
```

### 자동입찰 주말 규칙
- 금요일 기준: 전일(목) vs 전전일(수) 비교로 평상시와 동일하게 작성
- 토·일요일 기준: **"주말 저소진 구간 — 설정 변경 없이 검색 수요 감소에 따른 자연 감소"** 문구로 일괄 처리
- 노출순위는 주말에도 추이 기재

### 빠른 판별 기준
- 데이터 날짜가 **금(Fri) · 토(Sat) · 일(Sun)** → 무조건 주말 통합 규칙 적용
- 대시보드에서 해당 날짜 선택 시 자동으로 **노란색 강조 + "주말 합산" 배너** 표시됨

## 시트별 코멘트 양식

### 1. Summary (전체 요약)

**제목 형식:**
```
{날짜} ({요일}) 요약 — 전일({전일 날짜} {요일}) 대비
```

**본문 구조 (4줄):**

| 줄 | 내용 |
|---|---|
| 1 | 네이버 SA PC 주말 광고비/전환/CPA + 전주·전전주 비교 |
| 2 | 네이버 SA MO 주말 광고비/전환/CPA + 전주 비교 + 광고비 소진 패턴 |
| 3 | 주말 광고비 저소진 주요 키워드 + 원인 분석 |
| 4 | 구글 SA 브랜드·경쟁사·일반 CPA 요약 + 주요 비효율 포인트 |

**작성 예시 (4/19 주말 기준):**
```
네이버 SA PC: 이번주말 광고비 13.4만원 / 전환 8건 / CPA 16,757원 — 전주(14,307원) 대비 소폭 상승, 전전주(26,124원) 대비 개선. 노출·클릭 규모는 전주와 유사하게 유지.

네이버 SA MO: 이번주말 광고비 177.4만원 / 전환 47건 / CPA 37,743원 — 전주(55,777원) 대비 32% 개선, 전전주(50,006원) 대비도 개선. 다만 광고비는 전주(228.7만원) 대비 22% 감소. 검색 수요 감소에 따른 자연 저소진 패턴.

주말 광고비 저소진 주요 키워드 (평일 4/14~16 일평균 대비): [PC 입주청소] -61%, [MO 입주청소] -47%, [MO 입주청소가격] -46%, [원룸입주청소] MO -75%, [PC 입주청소비용] -78%. 자동입찰 설정 변경 없이 주말 검색 수요 감소에 따른 구조적 현상으로 판단.

구글 SA: 브랜드 CPA 37,596원(전주 6,462원 대비 상승), 경쟁사 CPA 44,347원(전주 75,093원 대비 개선), 일반 CPA 107,998원(전주 82,896원 대비 상승). 일반 캠페인 비효율 심화 — [입주청소] 주말 광고비 43만원 소진, 전환 0건.
```

### 2. 네이버 SA · PC

**제목 형식:**
```
네이버 SA · PC — {날짜} ({요일}) 전일({전일} {요일}) 대비
```

**본문 구조 (3줄):**

| 줄 | 내용 |
|---|---|
| 1 | 당일 광고비/전환/CPA + 전일 대비 + 목표 이내/초과 |
| 2 | 평일일평균 대비 소진율 + 주요 저소진 키워드 |
| 3 | 전환 발생 키워드 전환건수/CPA |

**작성 예시:**
```
광고비 44,550원(-83%) — 전일(35,102원) 대비 소폭 증가. 전환 3건, CPA 14,850원으로 목표 이내.

주말 PC 광고비 평일일평균(94,443원) 대비 -53% 저소진. [입주청소] PC 평일일평균 3.1만원 → 주말 0.9만원(-68%)으로 주요 원인 확인.

[PC 입주청소] 전환 1건, CPA 9,916원으로 전환 연결. [아파트입주청소] PC 전환 1건, CPA 4,731원으로 소량 전환 발생.
```

### 3. 네이버 SA · MO

**제목 형식:**
```
네이버 SA · MO — {날짜} ({요일}) 전일({전일} {요일}) 대비
```

**본문 구조 (4줄):**

| 줄 | 내용 |
|---|---|
| 1 | 당일 광고비/전환/CPA + 전일 대비 + 평일일평균 대비 소진율 |
| 2 | 주요 저소진 키워드 광고비 비교 |
| 3 | 전환 0건 키워드 나열 |
| 4 | 전환 발생 키워드 전환건수/CPA |

**작성 예시:**
```
광고비 65.8만원, 전환 14건, CPA 46,991원 — 전일(31,189원) 대비 상승. MO 주말 광고비 평일일평균(79.9만원) 대비 -18% 저소진.

[MO 입주청소] 광고비 17.8만원, CPA 59,278원 — 평일일평균(24.1만원) 대비 -26% 소진 감소. [입주청소가격] 평일일평균 15.9만원 → 주말 9.7만원(-39%).

[원룸입주청소] MO 평일일평균 7,262원 → 주말 1,030원(-86%)으로 주말 검색량 급감 확인. [서울입주청소] [부산입주청소] 전환 0건 지속.

[이사청소] MO 전환 1건, CPA 41,039원. [아파트입주청소] MO 전환 1건, CPA 13,521원으로 이사 수요 소폭 확인.
```

### 7. 네이버 자동입찰

**제목 형식:**
```
네이버 자동입찰 키워드 인사이트 — {시작일}~{종료일}
```

**본문 구조 (키워드별 1줄씩):**

| 키워드 | 포함 내용 |
|---|---|
| [MO] 입주청소 | 전일 전환건수/CPA → 전전일 대비 변화 + 노출순위 추이 |
| [MO] 이사청소 | 전일 전환건수/CPA → 전전일 대비 변화 + 노출순위 추이 |
| [MO] 입주청소비용 | 전일 전환 여부/CPA → 전전일 대비 + 노출순위 추이 |
| [MO] 입주청소가격 | 전일 전환건수/CPA → 전전일 대비 + 노출순위 추이 |
| [MO] 입주청소전문 | 전일 전환건수/CPA → 전전일 대비 + 노출순위 추이 |
| [PC] 입주청소 | 전일 전환건수/CPA → 전전일 대비 + 노출순위 추이 |
| [PC] 이사청소 | 전일 전환건수/CPA → 전전일 대비 + 노출순위 추이 |
| [PC] 입주청소비용 | 전일 전환건수/CPA → 전전일 대비 + 노출순위 추이 |
| [PC] 입주청소가격·입주청소전문 | 전일 전환건수/CPA → 전전일 대비 + 노출순위 추이 + PC 효과 평가 |

**작성 규칙:**
- 비교 기준: **전일** vs **전전일** (당일 기준 하루 전 vs 이틀 전)
- 노출순위 추이: 전전일 → 전일 방향으로 기재 (예: `3.2위 → 2.8위`)
- 전환건수와 CPA를 함께 언급 (전환 0건이면 CPA 생략)

**작성 예시:**
```
[MO] 입주청소 — 전일(4/21) 전환 3건, CPA 52,140원 — 전전일(4/20) 전환 5건, CPA 38,276원 대비 전환 감소·CPA 상승. 노출순위 2.1위 → 2.4위로 소폭 하락.

[MO] 이사청소 — 전일(4/21) 전환 1건, CPA 41,039원 — 전전일(4/20) 전환 0건 대비 전환 발생. 노출순위 1.8위 → 1.9위로 유사 유지.

[MO] 입주청소비용 — 전일(4/21) 전환 0건 지속. 전전일(4/20)도 전환 0건. 노출순위 1.3위 → 1.4위로 안정적 유지.

[MO] 입주청소가격 ★신규(4/17~) — 전일(4/21) 전환 1건, CPA 48,327원 — 전전일(4/20) 전환 2건, CPA 37,841원 대비 효율 하락. 노출순위 1.2위 → 1.1위로 개선.

[MO] 입주청소전문 ★신규(4/17~) — 전일(4/21) 전환 0건. 전전일(4/20)도 전환 0건 지속. 노출순위 1.1위 → 1.1위로 최상위 유지 중이나 전환 연결 미흡.

[PC] 입주청소 — 전일(4/21) 전환 1건, CPA 9,916원 — 전전일(4/20) 전환 3건, CPA 8,476원 대비 전환 감소. 노출순위 2.5위 → 2.8위로 소폭 하락.

[PC] 이사청소 — 전일(4/21) 전환 0건 — 전전일(4/20) 전환 1건, CPA 4,078원 대비 전환 미발생. 노출순위 3.1위 → 3.4위.

[PC] 입주청소비용 — 전일(4/21) 전환 1건, CPA 6,310원 — 전전일(4/20) 전환 0건 대비 개선. 노출순위 2.9위 → 3.0위로 유사 유지.
```
