# 크린토피아 홈클리닝 Daily Report 코멘트 작성 가이드

---

## 작성 원칙

- 보고서체 단문 (존댓말 금지)
- 키워드는 반드시 [ ] 대괄호 표기
- 수치는 구체적으로 (만원 단위, % 단위)
- "대폭" 표현 금지 → "상승" "개선" "감소" "초과" 등 담백하게
- 목표 CPA: 30,000원 기준으로 판단
- 자동입찰은 주말에 설정 변경 없음 (주말 저소진 = 검색 수요 감소로 판단)

---

## 시트별 코멘트 양식

---

### 1. Summary (전체 요약)

**제목 형식:**
```
{날짜} ({요일}) 요약 — 전일({전일 날짜} {요일}) 대비
```

**본문 구조 (4줄):**

| 줄 | 내용 |
|---|---|
| 1 | 네이버 SA PC 주말 광고비/전환/CPA + 전주·전전주 비교 |
| 2 | 네이버 SA MO 주말 광고비/전환/CPA + 전주 비교 + 광고비 소진 패턴 |
| 3 | 주말 광고비 저소진 주요 키워드 + 원인 분석 |
| 4 | 구글 SA 브랜드·경쟁사·일반 CPA 요약 + 주요 비효율 포인트 |

**작성 예시 (4/19 주말 기준):**
```
네이버 SA PC: 이번주말 광고비 13.4만원 / 전환 8건 / CPA 16,757원 — 전주(14,307원) 대비 소폭 상승, 전전주(26,124원) 대비 개선. 노출·클릭 규모는 전주와 유사하게 유지.

네이버 SA MO: 이번주말 광고비 177.4만원 / 전환 47건 / CPA 37,743원 — 전주(55,777원) 대비 32% 개선, 전전주(50,006원) 대비도 개선. 다만 광고비는 전주(228.7만원) 대비 22% 감소. 검색 수요 감소에 따른 자연 저소진 패턴.

주말 광고비 저소진 주요 키워드 (평일 4/14~16 일평균 대비): [PC 입주청소] -61%, [MO 입주청소] -47%, [MO 입주청소가격] -46%, [원룸입주청소] MO -75%, [PC 입주청소비용] -78%. 자동입찰 설정 변경 없이 주말 검색 수요 감소에 따른 구조적 현상으로 판단.

구글 SA: 브랜드 CPA 37,596원(전주 6,462원 대비 상승), 경쟁사 CPA 44,347원(전주 75,093원 대비 개선), 일반 CPA 107,998원(전주 82,896원 대비 상승). 일반 캠페인 비효율 심화 — [입주청소] 주말 광고비 43만원 소진, 전환 0건.
```

### 2. 네이버 SA · PC

**제목 형식:**
```
네이버 SA · PC — {날짜} ({요일}) 전일({전일} {요일}) 대비
```

**본문 구조 (3줄):**

| 줄 | 내용 |
|---|---|
| 1 | 당일 광고비/전환/CPA + 전일 대비 + 목표 이내/초과 |
| 2 | 평일일평균 대비 소진율 + 주요 저소진 키워드 |
| 3 | 전환 발생 키워드 CVR/CPA |

**작성 예시:**
```
광고비 44,550원(-83%) — 전일(35,102원) 대비 소폭 증가. 전환 3건, CPA 14,850원으로 목표 이내.

주말 PC 광고비 평일일평균(94,443원) 대비 -53% 저소진. [입주청소] PC 평일일평균 3.1만원 → 주말 0.9만원(-68%)으로 주요 원인 확인.

[PC 입주청소] CVR 12.5%, CPA 9,916원으로 전환 연결. [아파트입주청소] PC CVR 100%, CPA 4,731원으로 소량 전환 발생.
```

### 3. 네이버 SA · MO

**제목 형식:**
```
네이버 SA · MO — {날짜} ({요일}) 전일({전일} {요일}) 대비
```

**본문 구조 (4줄):**

| 줄 | 내용 |
|---|---|
| 1 | 당일 광고비/전환/CPA + 전일 대비 + 평일일평균 대비 소진율 |
| 2 | 주요 저소진 키워드 광고비 비교 |
| 3 | 전환 0건 키워드 나열 |
| 4 | 전환 발생 키워드 CVR/CPA |

**작성 예시:**
```
광고비 65.8만원, 전환 14건, CPA 46,991원 — 전일(31,189원) 대비 상승. MO 주말 광고비 평일일평균(79.9만원) 대비 -18% 저소진.

[MO 입주청소] 광고비 17.8만원, CPA 59,278원 — 평일일평균(24.1만원) 대비 -26% 소진 감소. [입주청소가격] 평일일평균 15.9만원 → 주말 9.7만원(-39%).

[원룸입주청소] MO 평일일평균 7,262원 → 주말 1,030원(-86%)으로 주말 검색량 급감 확인. [서울입주청소] [부산입주청소] 전환 0건 지속.

[이사청소] MO 전환 1건(CPA 41,039원), [아파트입주청소] MO 전환 1건(CPA 13,521원)으로 이사 수요 소폭 확인.
```


| 2 | 주요 키워드 CVR/CPA |

**작성 예시:**

```
[크린토피아입주청소] CVR 16.67%, CPA 6,082원으로 전환 연결. [크린토피아청소] 광고비 20,800원 소진, 전환 0건.
```

### 5. 구글 SA · 경쟁사

**제목 형식:**
```
구글 SA · 경쟁사 — {날짜} ({요일}) 전일({전일} {요일}) 대비
```

**본문 구조 (3줄):**

| 줄 | 내용 |
|---|---|
| 1 | 당일 광고비/전환/CPA + 전일 대비 + 목표 초과 여부 |
| 2 | 전환 발생 키워드 CPA (최고 효율 키워드 포함) |
| 3 | 전환 0건 키워드 나열 |

**작성 예시:**
```
광고비 15.0만원, 전환 3건, CPA 50,000원 — 전일(85,766원) 대비 개선. 목표(30,000원) 초과.

[미소입주청소] CPA 79,316원(전환 1건), [숨고입주청소] CPA 34,569원(전환 1건). [숨고이사청소] CPA 10,204원(전환 1건)으로 경쟁사 내 최고 효율.

[미소이사청소] 광고비 15,513원, 전환 0건. [영구크린입주청소] [아정당입주청소] 전환 0건 지속.
```


| 3 | 전환 발생 키워드 CPA |

**작성 예시:**
```
광고비 34.0만원, 전환 1건, CPA 340,009원 — 전일(54,787원) 대비 급등. 목표(30,000원) 초과.

[입주청소] 광고비 15.5만원 소진, 전환 0건 — 주말 핵심 비효율 구간. [입주청소가격] 광고비 5.2만원, 전환 0건.

[이사청소] 전환 1건(CPA 22,065원)으로 일반 캠페인 내 유일 전환. [이사청소비용] [새집입주청소] 전환 0건.
```

### 7. 네이버 자동입찰

**제목 형식:**
```
네이버 자동입찰 키워드 인사이트 — {시작일}~{종료일}
```

**본문 구조 (키워드별 1줄씩):**

| 키워드 | 포함 내용 |
|---|---|
| [MO] 입주청소 | CPC 범위, 전환/CPA, 주말 저소진 원인 |
| [MO] 이사청소 | 전환/CPA, 특이점 |
| [MO] 입주청소비용 | 전환 여부, 노출순위 |
| [MO] 입주청소가격 | 광고비, 전환/CPA, 목표 대비 평가 |
| [MO] 입주청소전문 | 노출/CPC, 전환 여부, 순위 |
| [PC] 입주청소 | 최고 효율일 CPA, 전환 패턴 |
| [PC] 이사청소 | 전환/CPA, 볼륨 |
| [PC] 입주청소비용 | 전환/CPA |
| [PC] 입주청소가격·입주청소전문 | 노출순위, 전환 여부, PC 효과 평가 |

**작성 예시:**
```
[MO] 입주청소 — 4/17~19 CPC 303~557원으로 낮게 유지. 전환 3건(4/19)으로 주말 기준 전환 발생. CPA 59,278원으로 목표 초과. 주말 광고비는 평일일평균 대비 자연 감소(검색 수요 감소).

[MO] 이사청소 — 4/18 CPA 70,674원, 4/19 CPA 41,039원으로 이사 수요 전환 연결. 주말에도 안정적 전환 발생.

[MO] 입주청소비용 — 4/17~19 전환 0건 지속. 노출순위 1.3~1.4위 유지 중.

[MO] 입주청소가격 ★신규(4/17~) — 4/17~19 일 평균 광고비 9.0만원 소진. 4/17 CPA 74,233원, 4/19 CPA 48,327원으로 전환 발생. 목표 대비 높은 수준이나 신규 키워드 학습 기간으로 모니터링 필요.

[MO] 입주청소전문 ★신규(4/17~) — 4/17~19 노출 13~19만, CPC 339~921원이나 전환 0건. 노출순위 1.1위로 최상위 노출 중이나 클릭 전환 연결 미흡.

[PC] 입주청소 — 4/17 CPA 8,476원(전환 3건)으로 4월 내 최고 효율. 4/19 CPA 9,916원(전환 1건). 산발적 전환 패턴 지속.

[PC] 이사청소 — 4/18 CPA 4,078원(전환 1건)으로 전환 발생. 4/17, 4/19 전환 0건. 볼륨 소.

[PC] 입주청소비용 — 4/17 CPA 6,310원(전환 1건). 4/18~19 전환 0건.

[PC] 입주청소가격·입주청소전문 ★신규(4/17~) — PC 순위 4.5~7.0위로 하위 노출. 클릭 및 전환 거의 미발생. PC에서의 효과는 제한적.
```

## 빠른 참고 표현 모음

| 상황 | 표현 |
|---|---|
| CPA 목표 이내 | `목표(30,000원) 이내` |
| CPA 목표 초과 | `목표(30,000원) 초과` |
| 전환 없음 | `전환 0건 지속` |
| 광고비 감소 | `평일일평균 대비 -{%}% 저소진` |
| CPA 개선 | `전일 대비 개선` |
