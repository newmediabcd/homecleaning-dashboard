# 크린토피아 홈클리닝 Daily Report 코멘트 작성 가이드

---

## 작성 원칙

- 보고서체 단문 (존댓말 금지)
- 키워드는 반드시 [ ] 대괄호 표기
- 수치는 구체적으로 (만원 단위, % 단위)
- "대폭" 표현 금지 → "상승" "개선" "감소" "초과" 등 담백하게
- 목표 CPA: 30,000원 기준으로 판단
- 자동입찰은 주말에 설정 변경 없음 (주말 저소진 = 검색 수요 감소로 판단)

---

## ⚠️ 주말 3일 통합 규칙 (금·토·일)

> **데이터가 금요일·토요일·일요일에 해당하면, 단일 날짜가 아닌 금~일 3일 합산 기준으로 코멘트를 작성한다.**  
> **비교 기준은 전일이 아닌 전주 동일 주말(금~일) 3일 합산이다.**

### 적용 대상
Summary, 네이버 PC, 네이버 MO, 네이버 자동입찰, 구글 브랜드, 구글 경쟁사, 구글 일반 — **전 페이지 공통 적용**

### 핵심 규칙 요약

| 항목 | 평일 기준 | 주말 3일 통합 기준 |
|---|---|---|
| 분석 단위 | 당일 1일 | **이번주 금+토+일 합산** |
| 비교 기준 | **전일** 대비 | **전주 금+토+일 합산** 대비 |
| 광고비/전환/CPA | 당일 수치 | **3일 합산** 수치 |
| 저소진 분석 | 평일 일평균 대비 | 전주 주말 합산 대비 + 평일 일평균 대비 |
| 자동입찰 비교 | 전일 vs 전전일 | **전주 동일 요일** 대비 (금→전주 금, 토→전주 토, 일→전주 일) |

### 제목 형식 (주말 버전)
```
{섹션명} — 주말 합산 ({이번주 금 MM/DD}~{이번주 일 MM/DD}) / 전주 주말({전주 금 MM/DD}~{전주 일 MM/DD}) 대비
```

### 빠른 판별
- 데이터 날짜가 **금(Fri) · 토(Sat) · 일(Sun)** → 무조건 주말 통합 규칙 적용
- 대시보드에서 해당 날짜 선택 시 자동으로 **노란색 강조 + "주말 합산" 배너** 표시됨

---

## 시트별 코멘트 양식

각 섹션은 **평일 버전**과 **주말 버전**으로 구분한다.

---

### 1. Summary (전체 요약)

#### 평일 버전

**제목 형식:**
```
{날짜} ({요일}) 요약 — 전일({전일 날짜} {요일}) 대비
```

**본문 구조 (6줄):**

| 줄 | 내용 |
|---|---|
| 1 | 네이버 SA 전체 광고비/전환/CPA + 전일 대비 |
| 2 | 네이버 PC/MO 세부 광고비·CPA + 전일 대비 |
| 3 | **네이버 SA PC** 주요 소진 변화 키워드 (광고비 ± % 또는 전환 발생/미발생) |
| 4 | **네이버 SA MO** 주요 소진 변화 키워드 (광고비 ± % 또는 전환 발생/미발생) |
| 5 | **구글 SA** 주요 소진 변화 키워드 (비효율 또는 전환 발생 키워드) |
| 6 | 구글 SA 브랜드·경쟁사·일반 CPA 요약 |

#### 주말 버전

**제목 형식:**
```
주말 합산 ({이번주 금 MM/DD}~{이번주 일 MM/DD}) 요약 — 전주 주말({전주 금 MM/DD}~{전주 일 MM/DD}) 대비
```

**본문 구조 (6줄):**

| 줄 | 내용 |
|---|---|
| 1 | 네이버 SA PC 이번주말 합산 광고비/전환/CPA → **전주 주말 합산** 대비 변화 |
| 2 | 네이버 SA MO 이번주말 합산 광고비/전환/CPA → **전주 주말 합산** 대비 변화 + 소진 패턴 |
| 3 | **네이버 SA PC** 주말 저소진 주요 키워드 (평일 일평균 대비 %) + 원인 분석 |
| 4 | **네이버 SA MO** 주말 저소진 주요 키워드 (평일 일평균 대비 %) + 원인 분석 |
| 5 | **구글 SA** 주말 주요 소진 변화 키워드 (비효율 또는 전환 발생 키워드) |
| 6 | 구글 SA 브랜드·경쟁사·일반 주말 합산 CPA → 전주 주말 대비 + 주요 비효율 포인트 |

**작성 예시 (4/25~4/27 주말 기준, 전주 주말 4/18~4/20 대비):**
```
네이버 SA PC: 이번주말 합산 광고비 13.4만원 / 전환 8건 / CPA 16,757원 — 전주 주말(광고비 15.2만원, CPA 14,307원) 대비 광고비 -12%, CPA 소폭 상승. 노출·클릭 규모 전주와 유사.

네이버 SA MO: 이번주말 합산 광고비 177.4만원 / 전환 47건 / CPA 37,743원 — 전주 주말(광고비 228.7만원, CPA 55,777원) 대비 광고비 -22%, CPA 32% 개선. 검색 수요 감소에 따른 자연 저소진 패턴 유지.

네이버 SA PC 주요 소진 변화 키워드 (평일 4/21~23 일평균 대비): [입주청소] -61%, [입주청소비용] -78%, [이사청소] -45%. 자동입찰 설정 변경 없이 주말 검색 수요 감소에 따른 구조적 현상으로 판단.

네이버 SA MO 주요 소진 변화 키워드 (평일 4/21~23 일평균 대비): [입주청소] -47%, [입주청소가격] -46%, [원룸입주청소] -75%. 자동입찰 설정 변경 없이 주말 검색 수요 감소에 따른 구조적 현상으로 판단.

구글 SA 주요 소진 변화 키워드: [입주청소] 주말 합산 광고비 43만원, 전환 0건 — 주말 3일 연속 핵심 비효율 구간. [입주청소가격] 주말 합산 광고비 15.6만원, 전환 0건.

구글 SA: 이번주말 합산 브랜드 CPA 37,596원(전주 주말 6,462원 대비 상승), 경쟁사 CPA 44,347원(전주 주말 75,093원 대비 개선), 일반 CPA 107,998원(전주 주말 82,896원 대비 상승). 일반 캠페인 비효율 지속.
```

### 2. 네이버 SA · PC

#### 평일 버전

**제목 형식:**
```
네이버 SA · PC — {날짜} ({요일}) 전일({전일} {요일}) 대비
```

**본문 구조 (3줄):**

| 줄 | 내용 |
|---|---|
| 1 | 당일 광고비/전환/CPA + **전일** 대비 + 목표 이내/초과 |
| 2 | 평일일평균 대비 소진율 + 주요 저소진 키워드 |
| 3 | 전환 발생 키워드 전환건수/CPA |

**작성 예시:**
```
광고비 44,550원 — 전일(35,102원) 대비 소폭 증가. 전환 3건, CPA 14,850원으로 목표(30,000원) 이내.

PC 광고비 평일일평균(94,443원) 대비 -53% 저소진. [입주청소] PC 평일일평균 3.1만원 → 당일 0.9만원(-68%)으로 주요 원인 확인.

[PC 입주청소] 전환 1건 CPA 9,916원, [아파트입주청소] 전환 1건 CPA 4,731원으로 소량 전환 발생.
```

#### 주말 버전

**제목 형식:**
```
네이버 SA · PC — 주말 합산 ({이번주 금 MM/DD}~{이번주 일 MM/DD}) / 전주 주말({전주 금 MM/DD}~{전주 일 MM/DD}) 대비
```

**본문 구조 (3줄):**

| 줄 | 내용 |
|---|---|
| 1 | 이번주말 합산 광고비/전환/CPA → **전주 주말 합산** 대비 변화 + 목표 이내/초과 |
| 2 | 평일 일평균 대비 저소진율 + 주요 저소진 키워드 (주말→전주말 비교) |
| 3 | 주말 합산 전환 발생 키워드 전환건수/CPA |

**작성 예시 (4/25~4/27 주말, 전주 4/18~4/20 대비):**
```
이번주말 합산 광고비 13.4만원 / 전환 8건 / CPA 16,757원 — 전주 주말 합산(광고비 15.2만원, 전환 8건, CPA 14,307원) 대비 광고비 -12%, CPA 소폭 상승. 목표(30,000원) 이내 유지.

주말 PC 광고비 평일일평균(94,443원) 대비 -53% 저소진. [입주청소] PC 전주 주말(1.2만원) 대비 이번주말(0.9만원)으로 추가 감소, 검색 수요 감소 지속.

[PC 입주청소] 주말 합산 전환 3건 CPA 12,400원, [아파트입주청소] 전환 2건 CPA 6,800원으로 주말 전환 연결 확인.
```

### 3. 네이버 SA · MO

#### 평일 버전

**제목 형식:**
```
네이버 SA · MO — {날짜} ({요일}) 전일({전일} {요일}) 대비
```

**본문 구조 (4줄):**

| 줄 | 내용 |
|---|---|
| 1 | 당일 광고비/전환/CPA + **전일** 대비 + 평일일평균 대비 소진율 |
| 2 | 주요 저소진 키워드 광고비 비교 |
| 3 | 전환 0건 키워드 나열 |
| 4 | 전환 발생 키워드 전환건수/CPA |

**작성 예시:**
```
광고비 65.8만원, 전환 14건, CPA 46,991원 — 전일(31,189원) 대비 상승. MO 광고비 평일일평균(79.9만원) 대비 -18% 저소진.

[MO 입주청소] 광고비 17.8만원 CPA 59,278원 — 평일일평균(24.1만원) 대비 -26% 감소. [입주청소가격] 평일일평균 15.9만원 → 당일 9.7만원(-39%).

[원룸입주청소] MO 평일일평균 7,262원 → 당일 1,030원(-86%)으로 검색량 급감. [서울입주청소] [부산입주청소] 전환 0건 지속.

[이사청소] MO 전환 1건 CPA 41,039원. [아파트입주청소] MO 전환 1건 CPA 13,521원으로 이사 수요 소폭 확인.
```

#### 주말 버전

**제목 형식:**
```
네이버 SA · MO — 주말 합산 ({이번주 금 MM/DD}~{이번주 일 MM/DD}) / 전주 주말({전주 금 MM/DD}~{전주 일 MM/DD}) 대비
```

**본문 구조 (4줄):**

| 줄 | 내용 |
|---|---|
| 1 | 이번주말 합산 광고비/전환/CPA → **전주 주말 합산** 대비 변화 + 평일일평균 대비 소진율 |
| 2 | 주요 저소진 키워드 (이번주말 합산 vs 전주 주말 합산 또는 평일일평균) |
| 3 | 전환 0건 키워드 (주말 합산 기준) |
| 4 | 전환 발생 키워드 주말 합산 전환건수/CPA |

**작성 예시 (4/25~4/27 주말, 전주 4/18~4/20 대비):**
```
이번주말 합산 광고비 190.4만원 / 전환 42건 / CPA 45,333원 — 전주 주말 합산(광고비 228.7만원, 전환 41건, CPA 55,777원) 대비 광고비 -17%, CPA 18% 개선. MO 주말 평균 광고비 63.5만원 — 평일일평균(79.9만원) 대비 -21% 저소진 구조 유지.

[MO 입주청소] 이번주말 합산 52.3만원(전주 주말 61.2만원 대비 -15%). [MO 입주청소가격] 이번주말 합산 28.4만원(전주 주말 34.1만원 대비 -17%). 주말 검색 수요 감소 지속으로 자동입찰 설정 변경 없음.

[원룸입주청소] MO 주말 합산 전환 0건 지속. [서울입주청소] [부산입주청소] 주말 3일 합산 전환 0건.

[MO 이사청소] 주말 합산 전환 4건 CPA 38,500원. [MO 아파트입주청소] 전환 3건 CPA 28,200원으로 이사·입주 수요 주말 전환 연결 확인.
```

### 4. 구글 SA · 브랜드

**제목 형식:**
```
구글 SA · 브랜드 — {날짜} ({요일}) 전일({전일} {요일}) 대비

| 줄 | 내용 |
|---|---|
| 1 | 당일 광고비/전환/CPA + **전일** 대비 + 목표 이내/초과 |
| 2 | 주요 키워드 전환건수/CPA |

**작성 예시:**
```
광고비 26,882원, 전환 1건, CPA 26,882원 — 전일(42,999원) 대비 개선. 목표(30,000원) 이내.

[크린토피아입주청소] 전환 1건 CPA 6,082원으로 전환 연결. [크린토피아청소] 광고비 20,800원 소진, 전환 0건.
```

#### 주말 버전

**제목 형식:**
```
구글 SA · 브랜드 — 주말 합산 ({이번주 금 MM/DD}~{이번주 일 MM/DD}) / 전주 주말({전주 금 MM/DD}~{전주 일 MM/DD}) 대비
```

**본문 구조 (2줄):**

| 줄 | 내용 |
|---|---|
| 1 | 이번주말 합산 광고비/전환/CPA → **전주 주말 합산** 대비 변화 + 목표 이내/초과 |

[크린토피아입주청소] 주말 합산 전환 2건 CPA 18,500원. [크린토피아청소] 광고비 3.8만원 소진, 전환 1건 CPA 38,000원으로 목표 소폭 초과.

### 5. 구글 SA · 경쟁사

**본문 구조 (3줄):**

### 7. 네이버 자동입찰

#### 평일 버전

**제목 형식:**
```
네이버 자동입찰 — {날짜} ({요일}) 전전일 대비
```

**본문 구조 (키워드별 1줄씩):**

| 키워드 | 포함 내용 |
|---|---|
| [MO] 입주청소 | **전일** 전환건수/CPA → **전전일** 대비 변화 + 노출순위 추이 |
| [MO] 이사청소 | 전일 전환건수/CPA → 전전일 대비 + 노출순위 추이 |
| [MO] 입주청소비용 | 전일 전환 여부/CPA → 전전일 대비 + 노출순위 추이 |
| [MO] 입주청소가격 | 전일 전환건수/CPA → 전전일 대비 + 노출순위 추이 |
| [MO] 입주청소전문 | 전일 전환건수/CPA → 전전일 대비 + 노출순위 추이 |
| [PC] 입주청소 | 전일 전환건수/CPA → 전전일 대비 + 노출순위 추이 |
| [PC] 이사청소 | 전일 전환건수/CPA → 전전일 대비 + 노출순위 추이 |
| [PC] 입주청소비용 | 전일 전환건수/CPA → 전전일 대비 + 노출순위 추이 |
| [PC] 입주청소가격·입주청소전문 | 전일 전환건수/CPA → 전전일 대비 + 노출순위 추이 + PC 효과 평가 |

**작성 규칙:**
- 비교 기준: **전일** vs **전전일** (당일 기준 하루 전 vs 이틀 전)
- 노출순위 추이: 전전일 → 전일 방향으로 기재 (예: `3.2위 → 2.8위`)
- 전환건수와 CPA를 함께 언급 (전환 0건이면 CPA 생략)

**작성 예시:**
```
[MO] 입주청소 — 전일(4/24) 전환 3건 CPA 52,140원 — 전전일(4/23) 전환 5건 CPA 38,276원 대비 전환 감소·CPA 상승. 노출순위 2.1위 → 2.4위로 소폭 하락.

[MO] 이사청소 — 전일(4/24) 전환 1건 CPA 41,039원 — 전전일(4/23) 전환 0건 대비 전환 발생. 노출순위 1.8위 → 1.9위로 유사 유지.

[MO] 입주청소비용 — 전일(4/24) 전환 0건 지속. 전전일(4/23)도 전환 0건. 노출순위 1.3위 → 1.4위로 안정적 유지.

[MO] 입주청소가격 — 전일(4/24) 전환 1건 CPA 48,327원 — 전전일(4/23) 전환 2건 CPA 37,841원 대비 효율 하락. 노출순위 1.2위 → 1.1위로 개선.

[MO] 입주청소전문 — 전일(4/24) 전환 0건 지속. 노출순위 1.1위 → 1.1위로 최상위 유지 중이나 전환 연결 미흡.

[PC] 입주청소 — 전일(4/24) 전환 1건 CPA 9,916원 — 전전일(4/23) 전환 3건 CPA 8,476원 대비 전환 감소. 노출순위 2.5위 → 2.8위로 소폭 하락.

[PC] 이사청소 — 전일(4/24) 전환 0건 — 전전일(4/23) 전환 1건 CPA 4,078원 대비 전환 미발생. 노출순위 3.1위 → 3.4위.

[PC] 입주청소비용 — 전일(4/24) 전환 1건 CPA 6,310원 — 전전일(4/23) 전환 0건 대비 개선. 노출순위 2.9위 → 3.0위로 유사 유지.

[PC] 입주청소가격·입주청소전문 — 전일(4/24) 전환 0건. 전전일(4/23)도 전환 0건. 노출순위 5.2위 → 6.0위로 하락. PC에서의 효과 제한적.
```

#### 주말 버전

**제목 형식:**
```
네이버 자동입찰 — 주말 합산 ({이번주 금 MM/DD}~{이번주 일 MM/DD}) / 전주 주말({전주 금 MM/DD}~{전주 일 MM/DD}) 대비
```

**본문 구조 (키워드별 1줄씩):**

| 키워드 | 포함 내용 |
|---|---|
| 각 키워드 MO/PC | 이번주말 합산 전환건수/CPA → **전주 동일 주말 합산** 대비 변화 + 노출순위 추이 |

**작성 규칙:**
- 비교 기준: **이번주 금~일 합산** vs **전주 금~일 합산**
- 노출순위: 이번주 금~일 평균 순위 vs 전주 금~일 평균 순위
- 전환 0건이 주말 3일 합산 기준으로도 0건이면 "주말 3일 전환 0건" 명시
- 자동입찰 설정 변경 없음 문구 공통 삽입

**작성 예시 (4/25~4/27 주말, 전주 4/18~4/20 대비):**
```

[MO] 이사청소 — 이번주말 합산 전환 3건 CPA 44,000원 — 전주 주말 합산(전환 2건 CPA 51,039원) 대비 전환 증가·CPA 개선. 노출순위 전주 평균 1.8위 → 이번주 1.7위로 소폭 개선.

[MO] 입주청소비용 — 이번주말 합산 전환 0건 — 전주 주말 합산도 전환 0건. 노출순위 전주 1.3위 → 이번주 1.4위. 주말 3일 전환 미발생 지속.

[MO] 입주청소가격 — 이번주말 합산 전환 2건 CPA 52,000원 — 전주 주말 합산(전환 3건 CPA 48,327원) 대비 전환 감소·CPA 상승. 노출순위 전주 1.2위 → 이번주 1.1위로 유사.

[MO] 입주청소전문 — 이번주말 합산 전환 0건 — 전주 주말도 전환 0건. 노출순위 1.1위 유지. 주말 3일 합산 전환 미발생 지속 모니터링 필요.

[PC] 입주청소 — 이번주말 합산 전환 4건 CPA 14,200원 — 전주 주말 합산(전환 3건 CPA 16,757원) 대비 전환 증가·CPA 개선. 노출순위 전주 평균 2.5위 → 이번주 2.3위로 소폭 개선.

[PC] 이사청소 — 이번주말 합산 전환 1건 CPA 9,800원 — 전주 주말 합산(전환 0건) 대비 전환 발생. 노출순위 전주 3.1위 → 이번주 3.2위로 유사.

[PC] 입주청소비용 — 이번주말 합산 전환 1건 CPA 7,200원 — 전주 주말 합산(전환 1건 CPA 6,310원) 대비 유사. 노출순위 2.9위 → 3.0위 안정적 유지.

[PC] 입주청소가격·입주청소전문 — 이번주말 합산 전환 0건 — 전주 주말도 전환 0건. 노출순위 전주 5.2위 → 이번주 5.8위로 소폭 하락. PC 주말 전환 효과 제한적.
